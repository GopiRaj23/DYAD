# DYAD

Watch YouTube together in perfect sync — no matter the distance.

## Features
- Synchronized YouTube playback for two people
- Two-way voice chat (peer-to-peer WebRTC)
- Emoji reactions that float over the video
- Shared play/pause controls — either person can control
- Queue next video from either side
- Mobile-first responsive design

## Run Locally

```bash
npm install
npm start
```

Open http://localhost:3000

## Tech Stack
- Node.js + Express
- Socket.io (real-time sync)
- YouTube IFrame API
- PeerJS (WebRTC voice)
